## 🚀 部署方式

### 方式一：Vercel 部署（推荐）

1. 推送代码到 GitHub
2. 在 Vercel 导入项目
3. 自动部署完成
4. 获得永久访问链接：`https://your-app.vercel.app`

### 方式二：本地演示

1. 启动开发服务器：`npm run dev`
2. 浏览器打开：`http://localhost:5173`
3. 按F12开启移动设备模拟（iPhone 12/13/14）

### 方式三：手机预览

1. 确保手机和电脑在同一WiFi
2. 查看电脑IP地址
3. 手机浏览器访问：`http://电脑IP:5173`

1. **页面空白**：检查控制台错误，刷新页面

3. **功能不工作**：检查localStorage是否启用
